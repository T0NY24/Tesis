# Tesis - Ingeniería TI - UIDE

Presentación interactiva de propuestas de Integración Curricular (TIC) para Ingeniería en Tecnologías de la Información.

## Proyectos Incluidos:
1. **Detector de Deepfakes**: Plataforma Forense con arquitectura Trinity.
2. **IA Avatar GAD**: Asistente virtual humano para el municipio.
3. **TiendaGenius (SaaS)**: Ecosistema para gestión de inventarios y auditoría.

## Tecnologías:
- HTML5 / CSS3 (Animaciones avanzadas & Glassmorphism)
- JavaScript Vanilla
- FontAwesome Icons
- Google Fonts (Outfit)
